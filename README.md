# Star Wars Database

Website design for displaying Star Wars data.

## How to run localy

1. Clone the repository
```bash
git clone git@github.com:UmberBn/star-wars-database.git
```
2. Go to project folder
```bash
cd star-wars-database
```
2. Install project dependencies
```bash
yarn
```
3. Run project
```bash
yarn start
```
